# docs/decisions/

Durable design decisions for this project.

A good decision note includes:
- **Context** — why this decision was needed
- **Decision** — what was chosen
- **Alternatives considered**
- **Consequences** — what this enables or forecloses

## Decisions

- `001-data-sources.md` — why GH Archive + GitHub API (not Octoverse, Innovation Graph, etc.)
- `002-cohort-classification.md` — how the four cohorts are defined and why
- `003-viz-stack.md` — why Observable Plot + static site (no framework, no build step)
