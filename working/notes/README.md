# working/notes/

Temporary notes, drafts, or rough analysis live here.

Promote anything durable to:
- `STATUS.md` — if it's current project state
- `docs/decisions/` — if it's a design decision
- `skills/` — if it's a repeatable procedure
- `pipeline/config.py` — if it's a threshold or parameter

## Examples of what belongs here

- Notes from a BigQuery exploration session
- Draft cohort threshold analysis before promoting to the tuning notebook
- Scratch calculations on quota usage
- Observations from inspecting the classified_users output
